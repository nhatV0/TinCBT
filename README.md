📘 Hệ Thống Học Liệu Tin Học THPTChào mừng thầy đến với kho lưu trữ mã nguồn của hệ thống học tập trực tuyến. Hệ thống này được thiết kế theo kiến trúc Module giúp việc cập nhật bài giảng trở nên cực kỳ đơn giản mà không cần can thiệp sâu vào giao diện.📂 Cấu trúc các File chínhHệ thống của thầy gồm 3 tầng file chính:1. Tầng Giao diện (Trang chủ chính)Tên file: index.htmlNhiệm vụ: Đây là "bộ não" xử lý giao diện, màu sắc, trình phát video và logic nạp dữ liệu.Lưu ý: Thầy không cần chỉnh sửa file này trừ khi muốn thay đổi giao diện lớn (đổi màu, đổi logo).2. Tầng Điều hướng (Trạm trung chuyển)Tên file: data.jsNhiệm vụ: Quản lý danh mục 9 khóa học hiển thị ngoài trang chủ.Khi nào cần sửa: Khi thầy muốn đổi tên khóa học, đổi icon, hoặc thay đổi đường dẫn đến file dữ liệu bài tập của môn đó.3. Tầng Nội dung (Các vệ tinh bài học)Đây là nơi chứa bài giảng chi tiết của từng môn học. Mỗi file tương ứng với một khóa học:tin10.js: Bài giảng lớp 10.tin11.js: Bài giảng lớp 11.cpp10.js: Bài giảng lập trình C++.(Và các file tương ứng khác đã khai báo trong data.js)Nhiệm vụ: Chứa tiêu đề bài học, mã Video Youtube, mô tả và link tài liệu.Đây là nơi thầy sẽ làm việc thường xuyên nhất.🛠 Hướng dẫn cập nhật bài giảng mớiĐể thêm một bài giảng vào môn Tin học 10, thầy thực hiện các bước sau:Mở file tin10.js trên GitHub.Nhấn vào biểu tượng Edit (Cái bút).Thêm một cụm bài học mới vào trong danh sách lessons theo mẫu:{ 
    title: "Tên bài học mới", 
    videoId: "Mã_Youtube", 
    desc: "Mô tả nội dung bài học", 
    downloadUrl: "Link_tài_liệu_PDF_nếu_có" 
}
Nhấn Commit changes để lưu lại. Website sẽ tự cập nhật sau khoảng 30 giây.⚠️ Lưu ý quan trọngvideoId: Chỉ lấy phần mã sau dấu = của link Youtube. Ví dụ: youtube.com/watch?v=dQw4w9WgXcQ -> Mã là dQw4w9WgXcQ.Đồng bộ tên file: Tên file trên GitHub (ví dụ tin11.js) phải viết giống hệt (kể cả chữ hoa/thường) với tên đã khai báo trong mục dataPath của file data.js.Tài liệu: Nếu bài học không có tài liệu đi kèm, hãy để downloadUrl: null.Hệ thống được tối ưu hóa cho giáo viên để quản lý học liệu một cách chuyên nghiệp và khoa học.
